DISCORD SES BOTU - HAZIR PAKET

İçindekiler:
- index.js
- package.json
- .env
- README.txt

Ne yapar:
- Bot online görünür
- VOICE_CHANNEL_ID yazarsan açılır açılmaz o ses kanalına girer
- !gel yazarsan bulunduğun ses kanalına gelir
- Bağlantı koparsa tekrar bağlanmaya çalışır
- Host açık kaldığı sürece ses kanalında kalır

Senin yapacağın:
1) .env dosyasını aç
2) TOKEN yerine bot tokenini yaz
3) VOICE_CHANNEL_ID yerine ses kanal ID'sini yaz
4) Terminal aç
5) npm install
6) npm start

Komutlar:
- !ping
- !gel
- !çık

Önemli:
- Bu botun bilgisayar kapalıyken de çalışması için Render / Railway / VPS gibi bir hosta koyman gerekir.
- GitHub'a .env dosyasını yükleme. Token gizli kalmalı.
